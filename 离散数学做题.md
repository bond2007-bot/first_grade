## 1. 有效公式（Well-Formed Formula, WFF）

### 判断方法

**有效公式（合式公式）** 是按照以下规则构造的合法命题公式：

| 规则          | 说明                                          |
| ----------- | ------------------------------------------- |
| **① 原子公式**  | 命题变元（如 p, q, r）是合式公式                        |
| **② 否定式**   | 若 A 是合式公式，则 ¬A 也是合式公式                       |
| **③ 二元联结式** | 若 A、B 是合式公式，则 (A∧B)、(A∨B)、(A→B)、(A↔B) 是合式公式 |
| **④ 有限次应用** | 只有有限次使用①②③得到的才是合式公式                         |

**判断要点：**
- 联结词必须连接两个公式（¬除外）
- 括号必须配对
- 不能出现两个联结词相邻（如 p∧∨q ❌）
- 不能以联结词开头或结尾（除 ¬ 外）

**示例：**
| 公式 | 是否有效 | 说明 |
|------|---------|------|
| `(p ∧ q) → r` | ✅ | 正确 |
| `p ∧ ∨ q` | ❌ | 联结词相邻 |
| `→ p` | ❌ | 以联结词开头 |
| `(p → (q ∨ r))` | ✅ | 正确 |
| `p q` | ❌ | 缺少联结词 |

## 2. 偏序、全序、良序、拟序

### 2.1 偏序关系（Partial Order）

**定义**：集合 A 上的关系 R，满足 **自反、反对称、传递**。

```
条件：
① 自反性：∀x∈A，xRx
② 反对称性：xRy ∧ yRx ⇒ x = y
③ 传递性：xRy ∧ yRz ⇒ xRz
```

**记作**：`≤`（通常用 `≼` 表示）
**示例**：实数上的 ≤、集合的 ⊆、正整数上的整除 |

### 2.2 全序关系（Total Order / Linear Order）

**定义**：偏序关系 + **可比性**（任意两个元素都可以比较）。

```
偏序 + ∀x,y∈A，xRy 或 yRx 至少成立一个
```

**示例**：实数上的 ≤（任意两个实数都可比）
**对比**：正整数上的整除 | 是偏序但不是全序（2和3不可比）

### 2.3 良序关系（Well Order）

**定义**：全序关系 + **每个非空子集都有最小元**。

```
全序 + A的任意非空子集都有最小元
```

**示例**：自然数集 N 上的 ≤ 是良序（非负整数集总有最小值）
**反例**：整数集 Z 上的 ≤ 是全序但不是良序（负数集没有最小值）

### 2.4 拟序关系（Quasi-Order / Preorder）

**定义**：满足 **自反** 和 **传递** 的关系（不要求反对称）。

```
① 自反性：∀x∈A，xRx
② 传递性：xRy ∧ yRz ⇒ xRz
```

**示例**：同一类型的推理关系，知识的蕴涵关系

### 对比总结

| 关系类型 | 自反 | 反对称 | 传递 | 可比性 | 最小元存在 |
|---------|:---:|:-----:|:---:|:-----:|:---------:|
| **偏序** | ✅ | ✅ | ✅ | ❌ | ❌ |
| **全序** | ✅ | ✅ | ✅ | ✅ | ❌ |
| **良序** | ✅ | ✅ | ✅ | ✅ | ✅ |
| **拟序** | ✅ | ❌ | ✅ | ❌ | ❌ |

## 4. 主合取范式 与 主析取范式

### 4.1 基本概念

| 概念 | 含义 | 示例 |
|------|------|------|
| **文字** | 命题变元或其否定 | p, ¬p |
| **简单析取式（子句）** | 文字的析取 | p ∨ ¬q ∨ r |
| **简单合取式（短语）** | 文字的合取 | p ∧ ¬q ∧ r |
| **析取范式（DNF）** | 简单合取式的析取 | (p∧q) ∨ (¬p∧r) |
| **合取范式（CNF）** | 简单析取式的合取 | (p∨q) ∧ (¬p∨r) |

### 4.2 主析取范式（Principal DNF）

**定义**：每个简单合取式都包含所有命题变元（或其否定），且每个变元**只出现一次**。

**求法步骤：**
1. 求公式的真值表
2. 找出**真值为 T** 的行
3. 每行对应一个极小项（变元为 T → 变元本身，F → 变元的否定）
4. 将这些极小项析取起来

**示例**：求 `(p → q) ∧ p` 的主析取范式
```
真值表：
p q | p→q | (p→q)∧p
T T |  T  |    T     → 极小项 p∧q
T F |  F  |    F
F T |  T  |    F
F F |  T  |    F

主析取范式：p ∧ q
```

### 4.3 主合取范式（Principal CNF）

**定义**：每个简单析取式都包含所有命题变元（或其否定），且每个变元**只出现一次**。

**求法步骤：**
1. 求公式的真值表
2. 找出**真值为 F** 的行
3. 每行对应一个极大项（变元为 F → 变元本身，T → 变元的否定）
4. 将这些极大项合取起来

**示例**：同上公式

```
真值表中 F 的行：
p=T,q=F → 极大项 ¬p∨q  (因为此时p为T取¬p，q为F取q)
p=F,q=T → 极大项 p∨¬q
p=F,q=F → 极大项 p∨q

主合取范式：(¬p ∨ q) ∧ (p ∨ ¬q) ∧ (p ∨ q)
```

### 4.4 快速判断

| 真值表结果     | 对应的范式           |     |
| --------- | --------------- | --- |
| 全部为 T（永真） | 主析取范式 = 所有极小项析取 |     |
| 全部为 F（永假） | 主合取范式 = 所有极大项合取 |     |
| 部分 T 部分 F | 两种范式均可求         |     |
                           ==**主析取要真 主合取要假**==

## 5. 哈斯图（Hasse Diagram）

### 5.1 绘制方法

绘制之前一定要判断是否满足偏序关系 即自反 反对称 传递

**哈斯图**是偏序关系的简化图表示，规则：
1. **去掉自环**（自反性隐含，不画）
2. **去掉传递边**（若有 a≤b≤c，只画 a-b 和 b-c，不画 a-c）
3. **方向向上**（若 a≤b，则 b 画在 a 的上方，箭头省略）

```
示例：A={1,2,3,6,9,18}，偏序关系为整除 |

        18
       /  \
      6    9
     / \  / 
    2   3     
     \ /      
      1        (1在底层，与所有数相连但用传递性省略)
     
简化哈斯图：
        18
       /  \
      6    9
     / \  / 
    2   3
     \ /
      1
```

### 5.2 各种"元"的确定
极小元和极大元一定存在 因为是找的suoyou
最小元和最大元不一定存在 因为是是找的唯一

上界和下界都是对于一个给定集合说的
并且上界和下界中都有可能包含所给子集中的元素
极大元和极小元是对于整个集合来说

| 概念      | 定义             | 在哈斯图上找法                    |
| ------- | -------------- | -------------------------- |
| **最小元** | 唯一元素 ≤ 所有元素    | 从底层往上找，**唯一**能到达所有顶点的最底层元素 |
| **最大元** | 唯一元素 ≥ 所有元素    | 从顶层往下找，**唯一**能到达所有顶点的最顶层元素 |
| **极小元** | 没有更小的元素（不唯一）   | 哈斯图最底层**所有**没有入边的元素        |
| **极大元** | 没有更大的元素（不唯一）   | 哈斯图最顶层**所有**没有出边的元素        |
| **上界**  | 元素 ≥ 给定子集中所有元素 | 子集的所有公共"祖先"                |
| **下界**  | 元素 ≤ 给定子集中所有元素 | 子集的所有公共"后代"                |
| **上确界** | 上界中的**最小元**    | 所有上界中最小的那个                 |
| **下确界** | 下界中的**最大元**    | 所有下界中最大的那个                 |

### 5.3 典型例题

```
哈斯图：
         18
       /    \
      6      9
     / \    / \
    2   3  3   (9的下层)
     \ /
      1

集合 A = {1,2,3,6,9,18}，R = 整除关系

求：子集 B = {2,3} 的各类元

上界：能被2和3整除的数 → {6,18}
上确界：6（上界中最小的）
下界：整除2和3的数 → {1}
下确界：1（下界中最大的）
极大/极小元以整个集合A来看：
  极小元：1（最底层）
  极大元：18（最顶层）
  最小元：1（1整除所有元素）
  最大元：18（所有元素整除18）
  注意：本例中最小元=极小元，最大元=极大元，是因为该偏序集有唯一的极小元和唯一的极大元
```

## 6. 符号化语句："当且仅当" 用 ↔ 表示

### 6.1 联结词对应表

| 汉语 | 逻辑联结词 | 符号 |
|------|-----------|:----:|
| 且、并且、同时 | 合取 | ∧ |
| 或、或者 | 析取 | ∨ |
| 非、不是、并非 | 否定 | ¬ |
| 如果…那么…、若…则… | 蕴含 | → |
| **当且仅当**、**等价** | **双蕴含** | **↔** |
| 只有…才… | 蕴含（倒过来） | ←（实际用 → 但条件调换） |

### 6.2 "当且仅当" 的符号化

```
"p 当且仅当 q" = (p → q) ∧ (q → p) = p ↔ q
```

**注意区分：**
| 语句 | 符号化 | 说明 |
|------|-------|------|
| p 当 q | q → p | "当"表示"如果" |
| **p 当且仅当 q** | **p ↔ q** | 双向蕴含 |
| p 仅当 q | p → q | "仅当"引出必要条件 |
| 只有 p 才 q | q → p | "只有…才"引出必要条件 |

**符号化技巧：**
- "如果 p 那么 q" = p → q（充分条件 → 必要条件）
- "p 当 q" = q → p（q是p的条件）
- "p 仅当 q" = p → q（q是p的必要条件）
- "p 当且仅当 q" = p ↔ q

## 7. 强连通分支

### 7.1 定义

**强连通**：在有向图中，若从顶点 u 到 v 和从 v 到 u **都有**路径，则称 u 和 v 强连通。

**强连通分支（SCC, Strongly Connected Component）**：**极大**的强连通子图（不能再加入任何其他顶点并保持强连通）。

```
示例有向图：
  a → b → c → d
  ↑   ↓   ↑   ↓
  ←── ←── ←──

强连通分支：
  {a, b}  →  a→b→c→a? 不，c不能回到a
  
  ┌─────────────────────┐
  │  a ←→ b ←→ c ←→ d  │  ← 整个环
  └─────────────────────┘
  若 a→b, b→c, c→a, c→d, d→c
  则 {a,b,c,d} 是一个SCC
```

### 7.2 求强连通分支的方法

**Kosaraju 算法：**
1. **DFS** 遍历原图，记录顶点完成顺序（后序入栈）
2. **转置图**（所有边反向）
3. 按栈顶顺序（后序的逆序）在转置图上做 DFS，每次 DFS 遍历到的顶点构成一个 SCC

**Tarjan 算法（更常用）：**
- 一次 DFS，用 lowlink 值判断 SCC
- 每个 SCC 对应一个 DFS 树上的子树

### 7.3 性质

- 强连通分支的**收缩图**（每个分支缩成一个点）是一个 **DAG（有向无环图）**
- DAG 一定有**拓扑排序**，因此 SCC 收缩图可拓扑排序

```
原图 → 收缩为DAG：
  
  原图：           收缩后：
  a→b→c→a          ┌───┐
  d→e→f→d          │SCC1│ → │SCC2│
  a→d              └───┘     └───┘
```

### 7.4 考试常考

- **判断强连通性**：每个顶点是否都能到达其他所有顶点
- **求 SCC 的数量和具体元素**：画图分析
- **收缩图的 DAG 性质**
- **与弱连通分支的区别**：
  - 弱连通：忽略方向后连通
  - 强连通：考虑方向后双向可达

## 8. 等价关系、等价类与商集

### 8.1 等价关系

**定义**：集合 A 上的关系 R，若同时满足 **自反、对称、传递**，则称 R 为等价关系。

```
① 自反性：∀x∈A，xRx
② 对称性：xRy ⇒ yRx
③ 传递性：xRy ∧ yRz ⇒ xRz
```

**示例**：
- 整数集上的 **模 m 同余关系** ≡ (mod m) 是等价关系
- 三角形集合上的 **全等关系** 是等价关系
- 人群中的 **同龄关系** 是等价关系

### 8.2 等价类

**定义**：设 R 是 A 上的等价关系，对于任意 a∈A，a 的等价类记为 [a]ᵣ（或简写为 [a]），是所有与 a 有等价关系的元素集合。

```
[a]ᵣ = { x ∈ A | xRa }
```

**示例**：整数集 Z 上模 3 同余关系 ≡ (mod 3)

```
[0] = { ..., -6, -3, 0, 3, 6, ... }  （余数为0）
[1] = { ..., -5, -2, 1, 4, 7, ... }  （余数为1）
[2] = { ..., -4, -1, 2, 5, 8, ... }  （余数为2）
```

### 8.3 商集

**定义**：等价关系 R 的所有等价类组成的集合称为 A 关于 R 的**商集**，记为 A/R。

```
A/R = { [a]ᵣ | a ∈ A }
```

**示例**：整数集 Z 上模 3 同余关系的商集

```
Z / ≡(mod 3) = { [0], [1], [2] }
```

### 8.4 划分与商集的关系

| 概念 | 定义 | 关系 |
|:----:|------|------|
| **划分** | 将集合 A 分成若干个**互不相交**的非空子集，且这些子集的并等于 A | 每种划分对应一个等价关系 |
| **商集** | 等价关系 R 的所有等价类组成的集合 | 商集本质上是 A 的一个划分 |

> **核心定理**：集合 A 上的等价关系与 A 的划分一一对应。
> - 已知等价关系 → 等价类构成一个划分（即商集）
> - 已知划分 → 定义"同一块中的元素相关"即得到等价关系

### 8.5 典型例题

```
题：设 A = {1,2,3,4}，R 是 A 上的等价关系，
   已知等价类：[1] = {1,2}，[3] = {3}，求商集 A/R。

解：
[1] = {1,2}，[3] = {3}
由于[4]还没确定，看4是否与某些元素等价。
等价类互不相交且并集为A，所以[4] = {4}

商集 A/R = { {1,2}, {3}, {4} }
```

```
题2（★★★ 由划分求等价关系）：设集合 A = {1,2,3,4,5,6}，
   给定 A 的一个划分 π = { {1,3,5}, {2,4}, {6} }，
   试由该划分确定 A 上的一个等价关系 R，并列出 R 中的所有序对。

解：
划分中的每个块就是一个等价类，同一块内的元素彼此等价。

三个等价类：
  [1] = [3] = [5] = {1,3,5}
  [2] = [4] = {2,4}
  [6] = {6}

等价关系 R 必须满足自反、对称、传递。
逐块列出所有序对：
块内自己和自己做卡氏积  然后再并集

  块 {1,3,5} → (1,1)(1,3)(1,5)(3,1)(3,3)(3,5)(5,1)(5,3)(5,5)
  块 {2,4}   → (2,2)(2,4)(4,2)(4,4)
  块 {6}     → (6,6)

R = { (1,1),(1,3),(1,5),(3,1),(3,3),(3,5),(5,1),(5,3),(5,5),
      (2,2),(2,4),(4,2),(4,4),
      (6,6) }

验证：
① 自反：∀x∈A，(x,x)∈R      ✅ （每个元素都与自己相关）
② 对称：若 (x,y)∈R 则 (y,x)∈R  ✅ （每对都是成对出现）
③ 传递：如 (1,3)∧(3,5)→(1,5)∈R ✅
```
> **核心对应关系**：**一个划分 ↔ 一个等价关系**
> - 划分的每个子集 = 一个等价类 → 同一子集内元素两两相关
> - 等价类的全体 = 商集

## 9. 欧拉图与哈密顿图的判断

### 9.1 欧拉图

#### 定义

| 概念 | 定义 |
|:----:|------|
| **欧拉通路** | 经过图中**每条边恰好一次**的通路 |
| **欧拉回路** | 经过图中**每条边恰好一次**的回路 |
| **欧拉图** | 含有**欧拉回路**的图 |
| **半欧拉图** | 含有**欧拉通路**但不含欧拉回路的图 |

#### 判定定理

| 图类型 | 判定条件 | 说明 |
|:------:|----------|------|
| **无向图存在欧拉回路** | **所有顶点度数均为偶数** | 充分必要条件 |
| **无向图存在欧拉通路** | **恰好两个顶点度数为奇数**（其余为偶数） | 充分必要条件，奇度顶点为端点 |
| **有向图存在欧拉回路** | **每个顶点的入度 = 出度** | 充分必要条件 |
| **有向图存在欧拉通路** | **一个顶点出度=入度+1**（起点）、**一个顶点入度=出度+1**（终点）、其余入度=出度 | 充分必要条件 |

```
示例：
  A ─── B ─── C
  │     │     │
  D ─── E ─── F

每个顶点度数：A(2) B(3) C(2) D(2) E(3) F(2)
奇数度顶点为 B 和 E → 存在欧拉通路（从B到E）
```

### 9.2 哈密顿图

#### 定义

| 概念 | 定义 |
|:----:|------|
| **哈密顿通路** | 经过图中**每个顶点恰好一次**的通路 |
| **哈密顿回路** | 经过图中**每个顶点恰好一次**的回路 |
| **哈密顿图** | 含有**哈密顿回路**的图 |

#### 判定条件（必要条件 vs 充分条件）

| 类型 | 条件 | 说明 |
|:----:|------|------|
| **必要条件**（无向） | 删去任意 k 个顶点，最多产生 k 个连通分支 | 用于**证明不是**哈密顿图 |
| **充分条件**（无向） | 任意两个不相邻顶点度数之和 ≥ n（顶点数） | **满足则一定是**哈密顿图（不满足不一定不是） |
| **充分条件**（有向） | 竞赛图（完全有向图）一定存在哈密顿通路 | |
删去元素个数为n的点集 若产生的连通分量个数大于n  则原图不是哈密顿图
#### 欧拉图 vs 哈密顿图对比

|   对比项    | 欧拉图          | 哈密顿图           |
| :------: | ------------ | -------------- |
| **关注点**  | **边**（每条边一次） | **顶点**（每个顶点一次） |
|  **判定**  | **充要条件**明确   | **没有简洁的充要条件**  |
|  **难易**  | 容易判断         | 较难判断           |
| **经典问题** | 七桥问题         | 旅行商问题          |

```
示例：判断下图是否为哈密顿图

  A ─── B ─── C
  │     │     │
  D ─── E ─── F

n=6，可找到哈密顿回路：A-B-C-F-E-D-A ✅ 是哈密顿图
```

## 10. 量词顺序对式子真值的影响

### 10.1 基本概念

| 量词 | 含义 | 符号 |
|:----:|------|:----:|
| **全称量词** | 所有、任意、每一个 | **∀** |
| **存在量词** | 存在、某个、至少有一个 | **∃** |

### 10.2 量词顺序的影响

**同种量词可交换顺序**，真值不变：
```
∀x∀y P(x,y)  ≡  ∀y∀x P(x,y)
∃x∃y P(x,y)  ≡  ∃y∃x P(x,y)
```

**不同量词不可交换顺序**，真值可能改变：
```
∀x∃y P(x,y)  ≠  ∃y∀x P(x,y)
```

### 10.3 经典例题

```
论域：整数集
谓词：P(x,y) 表示 x < y

① ∀x∃y P(x,y)：对任意整数x，存在整数y使得 x < y
   → 真 ✅（对每个x，取 y=x+1 即可）

② ∃y∀x P(x,y)：存在整数y，对任意整数x都有 x < y
   → 假 ❌（不存在比所有整数都大的整数）

结论：∀x∃y 与 ∃y∀x 含义不同，真值也不同！
```

### 10.4 常见题型

```
题1：设论域为实数集，P(x,y): x + y = 0，判断真值

① ∃x∀y P(x,y)：存在实数x，对所有实数y都有 x+y=0
   → 假 ❌（不存在与所有数相加都等于0的数）

② ∀y∃x P(x,y)：对所有实数y，存在实数x使得 x+y=0
   → 真 ✅（取 x = -y 即可）

题2：设论域为人类，判断下列语句的符号化
"所有人都会死亡"        → ∀x 会死(x)
"有人会死亡"            → ∃x 会死(x)
"每个人都有母亲"        → ∀x ∃y 母亲(y,x)
"存在一个人的母亲是所有人" → ∃y ∀x 母亲(y,x)  ❌（显然假的）

```

题3（★★★ 三个约束变量经典题）：设论域为整数集 Z，
  谓词 P(x,y,z): x + y = z，判断下列各式真值

① ∀x ∀y ∃z P(x,y,z)
   对任意整数x和y，都存在整数z使得 x + y = z
   → 真 ✅（取 z = x + y 即可，整数对加法封闭）

② ∀z ∃x ∃y P(x,y,z)
   对任意整数z，都存在整数x和y使得 x + y = z
   → 真 ✅（取 x = z, y = 0 或 x = 0, y = z 等）

③ ∃z ∀x ∀y P(x,y,z)
   存在整数z，使得对任意整数x和y都有 x + y = z
   → 假 ❌（不可能存在一个数等于所有x+y，取x=0,y=0得z=0，但x=1,y=0时1≠0）

④ ∃x ∃y ∀z P(x,y,z)
   存在整数x和y，使得对任意整数z都有 x + y = z
   → 假 ❌（x+y是固定值，不可能等于所有z）

⑤ ∀x ∃z ∀y P(x,y,z)
   对任意整数x，都存在整数z，使得对所有整数y都有 x + y = z
   → 假 ❌（z需同时等于x+0、x+1、x+(-1)...不可能）

⑥ ∃z ∀x ∃y P(x,y,z)
   存在整数z，使得对任意整数x，都存在整数y满足 x + y = z
   → 真 ✅（取z=0，则对任意x，取y=-x即可使得 x + (-x) = 0）

> 总结规律：**∃ 在 ∀ 前面时限制更强，∃ 在 ∀ 后面时更宽松**。
> 三个量词时，从**最外层量词**开始读，逐层理解约束范围。

### 10.5 量词顺序记忆口诀

> **"∀x∃y"**：每人都有……（每人都有母亲 ✅）
> **"∃y∀x"**：存在一个人对所有人都……（存在一个人的母亲是所有人 ❌）
> **不同量词，顺序不能调换！**

## 11. 通过邻接矩阵求两点之间的通路

### 11.1 基本定理

设图 G 的邻接矩阵为 A（n×n 矩阵），则：

| 矩阵 | 含义 |
|:----:|------|
| **A** | A[i][j] = 从 vᵢ 到 vⱼ **长度为 1** 的通路数 |
| **A²** | (A²)[i][j] = 从 vᵢ 到 vⱼ **长度为 2** 的通路数 |
| **A³** | (A³)[i][j] = 从 vᵢ 到 vⱼ **长度为 3** 的通路数 |
| **……** | …… |
| **Aᵏ** | (Aᵏ)[i][j] = 从 vᵢ 到 vⱼ **长度为 k** 的通路数 |

> **核心定理**：从顶点 vᵢ 到 vⱼ 长度为 k 的通路数目等于 (Aᵏ)[i][j]

### 11.2 可达矩阵

**可达矩阵 P** 表示图中任意两个顶点之间是否可达：

```
P = A + A² + A³ + ... + Aⁿ （布尔加法/逻辑加）

P[i][j] = 1 表示 vᵢ 到 vⱼ 可达（存在至少一条通路）
P[i][j] = 0 表示 vᵢ 到 vⱼ 不可达
```

### 11.3 手工计算示例

```
图：有向图
  v₁ ─→ v₂
   ↓     ↓
  v₃ ←── v₄

邻接矩阵 A：
     v₁ v₂ v₃ v₄
v₁  [0  1  1  0]
v₂  [0  0  0  1]
v₃  [0  0  0  0]
v₄  [0  0  1  0]

求 A²（长度为2的通路数）：
A² = A × A

     v₁ v₂ v₃ v₄
v₁  计算 v₁行×各列：
    A²[1][1]=0×0+1×0+1×0+0×0=0
    A²[1][2]=0×1+1×0+1×0+0×0=0
    A²[1][3]=0×1+1×0+1×0+0×1=0
    A²[1][4]=0×1+1×1+1×0+0×0=1  → v₁→v₂→v₄ ✅
v₂  A²[2][4]=0×1+0×1+0×0+1×0=0 
    ...（逐行计算）
    
结果 A²：
     v₁ v₂ v₃ v₄
v₁  [0  0  0  1]   ← v₁→v₂→v₄
v₂  [0  0  1  0]   ← v₂→v₄→v₃
v₃  [0  0  0  0]
v₄  [0  0  0  0]
```

### 11.4 常用公式总结

| 问题 | 公式 |
|:----:|------|
| 从 vᵢ 到 vⱼ 长度为 k 的通路数 | **(Aᵏ)[i][j]** |
| 从 vᵢ 到 vⱼ 长度 ≤ k 的通路数 | **(A + A² + ... + Aᵏ)[i][j]** |
| 图中长度为 k 的通路总数 | **Aᵏ 中所有元素之和** |
| 图中长度为 k 的回路总数 | **Aᵏ 中对角线元素之和** |
| 可达矩阵 | **P = A ∨ A² ∨ ... ∨ Aⁿ**（布尔运算） |

### 11.5 典型例题

题：已知有向图邻接矩阵 A，求 v₁ 到 v₃ 长度为 3 的通路数

     v₁ v₂ v₃
v₁  [0  1  0]
v₂  [0  0  1]
v₃  [1  0  0]

计算 A³：
A² = A × A = 
v₁: [0 0 1]  (v₁→v₂→v₃)
v₂: [1 0 0]  (v₂→v₃→v₁)
v₃: [0 1 0]  (v₃→v₁→v₂)

A³ = A² × A =
v₁: [1 0 0]  (v₁→v₂→v₃→v₁)
v₂: [0 1 0]  (v₂→v₃→v₁→v₂)
v₃: [0 0 1]  (v₃→v₁→v₂→v₃)

∴ v₁ 到 v₃ 长度为 3 的通路数为：A³[1][3] = 0
v₁ 到 v₁ 长度为 3 的回路数为：A³[1][1] = 1 ✅


### 11.6 注意事项

> ① Aᵏ 中的数值是**通路数目**（可能大于1），不是布尔值
> ② 对于无向图，邻接矩阵是**对称矩阵**
> ③ 对角线元素 Aᵏ[i][i] 表示从 vᵢ 出发回到 vᵢ 的回路数
> ④ 手工计算时建议逐行逐列计算，避免遗漏
