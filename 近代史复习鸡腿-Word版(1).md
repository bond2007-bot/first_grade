# 近代史复习鸡腿

---

## 上篇综述

### 1. 为什么说鸦片战争是中国近代史的起点

| 维度 | 变化 |
|------|------|
| **社会性质** | 独立封建→**半殖民地半封建社会** |
| **主要矛盾** | 农民vs地主 → **帝国主义vs中华民族、封建主义vs人民大众** |
| **革命任务** | 反本国封建 → +**反对外国殖民侵略**，转为资产阶级民族民主革命 |
| **发展方向** | 资本主义进程中断 → 转向**社会主义道路** |

---

### 2. 近代中国两大历史任务及其相互关系

| 任务 | 内容 | 关系 |
|:---:|------|------|
| **一** | 求得**民族独立和人民解放** | → 创造前提，开辟道路 |
| **二** | 实现**国家富强和人民富裕** | → 最终目的和必然要求 |

> 改变腐朽制度→争取民族独立解放→为实现国家富强创造前提；后者是前者的最终目的

---

### 3. 近代中国的主要矛盾、社会性质与基本特征

#### 主要矛盾
- **帝国主义和中华民族的矛盾**（最主要的矛盾）
- **封建主义和人民大众的矛盾**

#### 社会性质：半殖民地半封建社会

#### 基本特征

| # | 特征 |
|:-:|------|
| ① | **资本—帝国主义**操纵中国财政经济命脉和政治 |
| ② | **封建势力**衰败并与外国侵略势力相勾结 |
| ③ | **封建地主土地所有制**仍是封建剥削制度的根基 |
| ④ | **民族资本主义发展缓慢**，力量软弱 |
| ⑤ | 各地区经济政治文化**发展极不平衡** |
| ⑥ | 广大人民特别是农民**贫困化、饥寒交迫** |

#### 各领域特征对比

| 领域 | 特征 |
|------|------|
| **经济** | 封建地主土地所有制经济占主导 |
| **政治** | 高度中央集权的封建君主专制制度 |
| **社会结构** | 族权+政权结合的封建宗法等级制度（核心：宗族家长制） |
| **文化思想** | 儒家思想为正统，与佛道相互吸收融合 |

#### 阶级来源

| 阶级 | 来源 |
|------|------|
| **无产阶级（工人阶级）** | 城乡破产失业农民、手工业者、城市贫民（出现早于资产阶级） |
| **资产阶级** | 一部分为官僚买办资本家，另一部分为民族资本家 |

### 5. 鸦片战争等历次反侵略战争失败原因

| 因素 | 分析 |
|:---:|------|
| **根本原因（内因）** | **近代中国社会制度的腐败**。清政府政治腐败、经济落后、文化保守，没有应变能力，**压制破坏人民群众的反侵略斗争** |
| **重要原因（外因）** | **近代中国经济技术的落后**。英国已工业革命，中国仍停留在自然经济，军队不了解近代战术 |

---

### 6. 资本-帝国主义侵略中国的主要手段

| 手段 | 具体内容 |
|:---:|----------|
| **军事侵略** | 发动战争、屠杀人民、侵占领土、勒索赔款、抢掠财富 |
| **政治统治** | 控制内政外交、镇压反抗、扶植收买代理人 |
| **经济掠夺** | 控制通商口岸、剥夺关税自由、商品倾销和资本输出、操控经济命脉 |
| **文化渗透** | 披宗教外衣进行侵略、制造侵略舆论 |

### 7. 太平天国起义的历史意义

| # | 意义 |
|:-:|------|
| ① | **沉重打击了封建统治阶级**，**加速了清王朝的衰败过程** |
| ② | 旧式农民战争的**最高峰** |
| ③ | 冲击了孔子和儒家经典的正统权威 |
| ④ | **有力打击了外国侵略势力**，拒绝承认不平等条约 |
| ⑤ | 19世纪中叶亚洲民族解放运动中**时间最久、规模最大、影响最深** |

### 8. 太平天国失败原因与教训

#### 主观原因

| # | 原因 |
|:-:|------|
| ① | 农民阶级不是新生产力代表，无法克服阶级局限性 |
| ② | 拜上帝教不是科学思想理论，不能正确指导战争 |
| ③ | 没能正确对待儒学 |
| ④ | 不能区分西方侵略者与人民群众 |

#### 客观原因：中外反动势力勾结起来联合镇压

#### 教训

> ① 半殖民地半封建的中国，农民具有伟大的革命潜力
> ② 但自身**不能肩负起**领导反帝反封建斗争取得胜利的重任
> ③ 单纯的农民战争不可能完成民族独立和人民解放的历史任务

**核心概念**：**中体西用**——"中学为体、西学为用"，以中国伦理纲常为主体，以西方近代技术为辅助。

| 方面 | 主要内容 |
|:---:|----------|
| **兴办近代企业** | 军用工业 + 民用企业 |
| **建立新式海陆军** | 配外国教练，装备洋枪洋炮 |
| **创办新式学堂、派遣留学生** | 翻译学堂、工艺学堂、军事学堂等 |

#### 洋务运动的历史作用

| # | 作用 |
|:-:|------|
| ① | 客观上对中国早期工业和民族资本主义发展有**促进作用** |
| ② | 最早官派留学生，是中国近代教育的开始，带来新知识 |
| ③ | 资本主义生产方式出现，冲击"重本抑末"观念 |

#### 洋务运动失败原因

| # | 原因 |
|:-:|------|
| ① | **封建性**——在不改变封建制度的前提下学西方技术 |
| ② | **对外依赖性**——处处依赖外国 |
| ③ | **管理腐朽性**——封建衙门式管理，营私舞弊 |

> **启示**：地主阶级不能担负起中国近代化的历史重任

## 中编综述
