## 一、平均查找长度（ASL）

```
ASL = Σ(Pi × Ci)

Pi：第i个元素的查找概率
Ci：查找第i个元素需要比较的次数
```

## 二、顺序查找（Sequential Search）

**思想**：从头到尾遍历查找。

```c
#include<stdio.h>
#include<stdlib.h>

int search(int* data, int len, int value)
{
    for(int i = 0; i < len; i++)
    {
        if(data[i] == value)
            return i;
    }
    return -1;  // 不返回0，因为位置可能为0
}
```

### 特性
| 项目 | 值 |
|------|-----|
| 时间复杂度 | O(n) |
| ASL成功 | (n+1)/2 |
| ASL失败 | n+1 |
| 适用场景 | 无序表、小规模数据 |
| **优化方式** | 设置"哨兵"存放于0号位置，简化边界判断 |

## 三、折半查找（Binary Search）

**前提**：查找表必须为**有序**，采用**顺序存储结构**（不适用于链表）。

### 算法思想
```
1. low=0, high=len-1
2. mid = (low + high) / 2（向下取整）
3. 判断：
   - value == a[mid] → 返回
   - value < a[mid] → high = mid - 1（找左半）
   - value > a[mid] → low = mid + 1（找右半）
4. 重复直到low > high（查找失败）
```

```c
int binary_search(int* data, int len, int value)
{
    int low = 0;
    int high = len - 1;
    int mid;

    while(low <= high)
    {
        mid = (low + high) / 2;
        if(data[mid] == value)
            return mid;
        else if(data[mid] < value)
            low = mid + 1;
        else
            high = mid - 1;
    }
    return -1;
}
```

### 执行过程示例
```
有序表：[2, 13, 22, 38, 46, 58, 66, 77]
查找58：

第1次：low=0, high=7, mid=3 → data[3]=38 < 58 → low=4
第2次：low=4, high=7, mid=5 → data[5]=58 == 58 ✅ 返回5
比较次数：2次
```

### 折半查找判定树

```
给定有序序列，可以画出一棵折半查找判定树：
- 根节点是mid位置元素
- 左子树是左半部分
- 右子树是右半部分

例如序列 [2, 13, 22, 38, 46, 58, 66, 77] 的判定树：
          46
        /    \
      22      66
     /  \    /  \
    13  38  58  77
   /
  2
```

### 特性
| 项目 | 值 |
|------|-----|
| 时间复杂度 | O(log₂n) |
| ASL成功 | ≈ log₂(n+1) - 1 |
| 空间复杂度 | O(1) |
| 前提条件 | 有序 + 顺序存储 |

> **折半查找判定树的高度**：h = ⌊log₂n⌋ + 1

## 四、二叉排序（搜索）树（BST）

**定义**：对任意一个节点，满足左子树上所有的值都小于根节点，右子树上所有的值都大于根节点。

```c
typedef int ElemType;

typedef struct TreeNode
{
    ElemType data;
    struct TreeNode* lchild;
    struct TreeNode* rchild;
}TreeNode;

typedef TreeNode* BiTree;
```

### 1. 二叉排序树的创建

```c
// 数组模拟先序创建二叉排序树，-1表示空节点
int treeArr[] = {70, 55, 49, 30, -1, -1, 53, -1, -1, -1, 80, 75, -1, -1, 98, 95, -1, -1, -1};
int idx = 0;

void createTree(BiTree* T)
{
    ElemType num = treeArr[idx++];
    if(num == -1)
    {
        *T = NULL;
    }
    else
    {
        *T = (BiTree)malloc(sizeof(TreeNode));
        (*T)->data = num;
        createTree(&(*T)->lchild);
        createTree(&(*T)->rchild);
    }
}
```

### 2. 先序遍历

```c
void preOrder(BiTree T)
{
    if(T == NULL) return;
    printf("%d ", T->data);
    preOrder(T->lchild);
    preOrder(T->rchild);
}
```

### 3. 二叉排序树的查找

```c
/*
 * 参数：
 * T: 当前子树的根节点
 * value: 要查找的值
 * parent: 当前节点的父节点
 * pos: 指针变量，返回找到的节点
 * 返回1找到，返回0没找到
 */
int search_bst(BiTree T, int value, BiTree parent, BiTree* pos)
{
    if(T == NULL)
    {
        *pos = parent;  // 查找失败，返回该空节点的父节点
        return 0;
    }
    if(T->data == value)
    {
        *pos = T;
        return 1;
    }
    else if(T->data > value)
        return search_bst(T->lchild, value, T, pos);
    else
        return search_bst(T->rchild, value, T, pos);
}
```

### 4. 二叉排序树的插入

```c
int insert_bst(BiTree* T, int value)
{
    BiTree pos, current;
    int status = search_bst(*T, value, NULL, &pos);

    if(status == 0)  // 不存在，可以插入
    {
        current = (BiTree)malloc(sizeof(TreeNode));
        current->data = value;
        current->lchild = NULL;
        current->rchild = NULL;

        if(pos == NULL)  // 树为空
            *T = current;
        else if(value < pos->data)
            pos->lchild = current;
        else
            pos->rchild = current;
        return 1;
    }
    return 0;  // 已存在，不插入
}
```

### 5. 二叉排序树的删除

```c
// 真正执行删除操作
int delete(BiTree* d)
{
    BiTree temp, record;

    if((*d)->rchild == NULL)     // 只有左子树或无子节点
    {
        temp = *d;
        *d = (*d)->lchild;
        free(temp);
    }
    else if((*d)->lchild == NULL)  // 只有右子树
    {
        temp = *d;
        *d = (*d)->rchild;
        free(temp);
    }
    else  // 有两个孩子：用左子树的最大值替换
    {
        temp = *d;
        record = (*d)->lchild;
        while(record->rchild != NULL)
        {
            temp = record;
            record = record->rchild;
        }
        (*d)->data = record->data;  // 替换数据
        if(temp != *d)
            temp->rchild = record->lchild;
        else
            temp->lchild = record->lchild;
        free(record);
    }
}

// 递归查找并删除
int delete_bst(BiTree* T, int value)
{
    if(*T == NULL)
    {
        printf("not found\n");
        return 0;
    }
    if((*T)->data == value)
        return delete(T);
    else if((*T)->data > value)
        return delete_bst(&(*T)->lchild, value);
    else
        return delete_bst(&(*T)->rchild, value);
}
```

### 6. 主函数测试

```c
int main()
{
    BiTree T = NULL;
    int treeArr[] = {70, 55, 49, 30, 39, 53, 80, 75, 98, 95};
    int len = sizeof(treeArr) / sizeof(treeArr[0]);

    for(int i = 0; i < len; i++)
        insert_bst(&T, treeArr[i]);

    preOrder(T); printf("\n");

    insert_bst(&T, 99);
    preOrder(T); printf("\n");

    delete_bst(&T, 49);
    preOrder(T); printf("\n");

    return 0;
}
```

### 重要性质
- **中序遍历二叉排序树得到递增有序序列**
- 构造二叉排序树的目的不仅仅是排序，而是提高查找、插入、删除的速度

## 五、平衡二叉树（AVL树）

**定义**：
- 前提是一个二叉排序树
- 任意一个节点的左子树和右子树的高度之差的**绝对值不超过1**
- **平衡因子** = 左子树高度 - 右子树高度

**平衡调整**：
- 最小不平衡子树：从插入点向上回溯，第一个失衡的节点的子树
- 四种旋转方式：**LL、RR、LR、RL**

---

## 六、折半查找与二叉排序树对比

| 对比项 | 折半查找 | 二叉排序树 |
|-------|---------|-----------|
| 前提条件 | 有序顺序表 | 二叉排序树结构 |
| 时间复杂度 | O(log₂n) | O(n)（最坏）/ O(log₂n)（最好） |
| 插入/删除 | O(n)（移动元素） | O(h)（h为树高） |
| 适用场景 | 静态查找（不常变动） | 动态查找（频繁插入删除） |

> **折半查找判定树**：若一个折半查找的顺序表的排列顺序与一个二叉排序树的中序遍历结果相同，则两者是对应的。

### 📌 常见题型
- **选择题**：时间复杂度、查找长度计算
- **填空题**：画出判定树、给定序列构建二叉排序树
- **简答题**：
  - 计算ASL
  - 二叉排序树删除指定节点后画出新树
  - 平衡二叉树构建过程（LL/RR/LR/RL旋转）
- **算法题**：二叉排序树的查找、插入、删除代码实现

### 📌 易错点
| 易错点 | 说明 |
|--------|------|
| 折半查找必须用顺序存储 | 链表不支持随机访问 |
| 二叉排序树删除分3种情况 | 叶子、单孩子、双孩子 |
| 删除双孩子节点要用前驱/后继替换 | 通常用左子树最大值或右子树最小值 |
| AVL的平衡因子 | 左高-右高，绝对值≤1 |
