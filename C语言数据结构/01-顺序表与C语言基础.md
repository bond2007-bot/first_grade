### 1. 指针与值交换

```c
#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>

/*通过指针交换两个变量的值*/
void swap(int* a, int* b)
{
    int temp;
    temp = *b;
    *b = *a;
    *a = temp;
    printf("a = %d,b = %d", *a, *b);
}

int main()
{
    int x = 5;
    int y = 6;
    swap(&x, &y);
    return 0;
}
```

## 二、顺序表（SeqList）

### 1. 顺序表定义与初始化（静态分配）

```c
#define MAXSIZE 5
typedef int ElemType;

typedef struct
{
    ElemType data[MAXSIZE];
    int length;
}SeqList;

void initList(SeqList *L)
{
    L->length = 0;
}
```

### 2. 顺序表动态分配内存

```c
typedef struct
{
    ElemType *data;  // 访问data元素时使用数组下标形式
    int length;
}SeqList;

SeqList* initList()
{
    SeqList* L = (SeqList*)malloc(sizeof(SeqList));
    L->data = (ElemType*)malloc(sizeof(ElemType)*MAXSIZE);
    L->length = 0;
    return L;
}
```

### 3. 尾部添加元素

```c
int appendElem(SeqList* L, ElemType e)
{
    if (L->length >= MAXSIZE)
    {
        printf("顺序表已满\n");
        return 0;
    }
    L->data[L->length] = e;
    L->length++;
    return 1;  // 函数中0表示失败，1表示成功
}
```

### 4. 遍历顺序表

```c
void listElem(SeqList* L)
{
    for(int i = 0; i < L->length; i++)
    {
        printf("%d ", L->data[i]);
    }
    printf("\n");
}
```

### 5. 插入元素（指定位置）

```c
int insertElem(SeqList* L, int pos, ElemType e)  // pos从1开始计数
{
    if(pos > L->length)
    {
        printf("无法插入，请检查插入的位置是否合理");
        return 0;
    }
    else
    {
        for(int i = L->length - 1; i >= pos - 1; i--)
        {
            L->data[i+1] = L->data[i];
        }
        L->data[pos-1] = e;
        L->length++;
    }
    return 1;
}
```

### 6. 删除元素

```c
int deleteElem(SeqList* L, int pos)
{
    if (pos < L->length)
    {
        for(int i = pos; i < L->length; i++)
        {
            L->data[i-1] = L->data[i];
        }
        L->length--;
        return 1;
    }
}
```

### 7. 查找元素

```c
int findElem(SeqList* L, ElemType e)
{
    if(L->length == 0)
    {
        printf("这是一个空列表\n");
        return 0;
    }
    int i;
    for(i = 0; i < L->length; i++)
    {
        if(L->data[i] == e)
        {
            return i+1;
        }
    }
    if(i == L->length)
    {
        printf("没有找到该元素");
    }
    return 0;
}
```

### 8. 主函数测试


## 三、顺序表 vs 链表对比（高频考点）

| 比较维度 | 顺序表 | 链表 |
|---------|-------|------|
| **随机访问** | O(1) ✅ | O(n) ❌ |
| **插入/删除** | O(n) ❌ | O(1) ✅（已知位置） |
| **空间分配** | 预分配，易浪费/溢出 | 动态分配，灵活 |
| **存储密度** | 高（无指针开销） | 低（需额外指针域） |
| **适用场景** | 频繁访问、少插入删除 | 频繁插入删除、长度变化大 |

## 四、期末考试考点

### 📌 常见题型
1. **选择题**：顺序表与链表的对比、动态内存分配、结构体使用
2. **填空题**：顺序表插入/删除操作的元素移动次数
3. **简答题**：比较顺序表和链表的优缺点及适用场景
4. **算法题**：顺序表的基本操作实现（插入、删除、查找）

### 📌 易错点
- 顺序表插入/删除时元素的移动方向（插入从后往前移，删除从前往后移）
- `malloc` 返回 `void*` 需要强制类型转换
- 使用完动态分配的内存后一定要 `free()`
- 结构体指针使用 `->` 访问成员，结构体变量使用 `.` 访问成员
