## 一、希尔排序（Shell Sort）

**思想**：插入排序的改进版，先让数据基本有序。按一定的步长分组排序，逐步减少步长，步长为1时完成最终排序。

```c
#include<stdio.h>

void shellSort(int* data, int len)
{
    int temp;
    int step = len / 2;  // 初始化步长为数组长度的一半

    while(step >= 1)
    {
        // 对每个分组进行插入排序
        for(int i = step; i < len; i++)
        {
            if(data[i] < data[i-step])
            {
                temp = data[i];
                int j = i - step;
                while(j >= 0 && temp < data[j])
                {
                    data[j + step] = data[j];
                    j = j - step;
                }
                data[j + step] = temp;
            }
        }
        step = step / 2;  // 缩小步长
    }
}
```

### 执行过程示例
```
初始：47 35 60 95 77 15 28 52

step=4: 分组(47,77) (35,15) (60,28) (95,52)
        结果：47 15 28 52 77 35 60 95

step=2: 分组(47,28,77,60) (15,52,35,95)
        结果：28 15 47 35 60 52 77 95

step=1: 直接插入排序
        结果：15 28 35 47 52 60 77 95
```

### 特性
| 项目 | 值 |
|------|-----|
| 平均时间复杂度 | O(n^1.3) |
| 最坏时间复杂度 | O(n²) |
| 空间复杂度 | O(1) |
| 稳定性 | ❌ **不稳定** |
| **特点** | 先分组排序，逐步缩小组距，最后全局有序 |

## 二、归并排序（Merge Sort）

**思想**：分治策略。将两个有序子序列合并，不断递归分割再合并。

```c
// 合并两个有序子序列：[left...mid] 和 [mid+1...right]
void merge(int* data, int left, int mid, int right)
{
    int temp[100];  // 辅助数组
    int i = left;
    int j = mid + 1;
    int k = 0;

    // 将两个有序序列按大小依次放入temp
    while(i <= mid && j <= right)
    {
        if(data[i] <= data[j])
            temp[k++] = data[i++];
        else
            temp[k++] = data[j++];
    }

    // 拷贝剩余部分
    while(i <= mid) temp[k++] = data[i++];
    while(j <= right) temp[k++] = data[j++];

    // 将temp中的结果拷贝回原数组
    for(int m = 0; m < k; m++)
        data[left + m] = temp[m];
}

// 归并排序递归函数
void mergeSort(int* data, int left, int right)
{
    if(left < right)  // 至少两个数据
    {
        int mid = (left + right) / 2;
        mergeSort(data, left, mid);       // 排序左边
        mergeSort(data, mid+1, right);    // 排序右边
        merge(data, left, mid, right);     // 合并
    }
}
```

### 执行过程示例
```
初始：47 35 60 95 77 15 28 52

分割：
[47 35 60 95] [77 15 28 52]
[47 35] [60 95] [77 15] [28 52]
[47] [35] [60] [95] [77] [15] [28] [52]

归并：
[35 47] [60 95] [15 77] [28 52]
[35 47 60 95] [15 28 52 77]
[15 28 35 47 52 60 77 95]
```

### 特性
| 项目 | 值 |
|------|-----|
| 时间复杂度 | O(n log n)（最坏/平均） |
| 空间复杂度 | **O(n)**（需要辅助数组） |
| 稳定性 | ✅ **稳定** |
| **特点** | O(n log n)级别中唯一稳定的排序 |

## 三、快速排序（Quick Sort）

**思想**：选一个基准元素（pivot），小的放左边，大的放右边，再递归对两个子序列进行同样操作。

```c
// 划分函数：将数组分为左右两部分，返回基准最终位置
int partition(int* data, int low, int high)
{
    int sta = data[low];  // 选左边元素作为基准值

    while(low < high)
    {
        // 从右往左找第一个比基准值小的数
        while(data[high] >= sta)
            high--;
        data[low] = data[high];  // 放左边

        // 从左往右找比基准值大的数
        while(data[low] <= sta)
            low++;
        data[high] = data[low];  // 放右边
    }
    data[low] = sta;  // 基准归位
    return low;
}

void QuickSort(int* data, int low, int high)
{
    int pivot;
    if(low < high)
    {
        pivot = partition(data, low, high);   // 划分
        QuickSort(data, low, pivot - 1);      // 排序左边
        QuickSort(data, pivot + 1, high);     // 排序右边
    }
}
```

### 执行过程示例（以第一个元素为基准）
```
初始：47 35 60 95 77 15 28 52

第1趟：基准=47
       ←从右找小：28  →从左找大：60
       28 35 [47] 95 77 15 60 52   ← 47归位
       28 35 [47] 95 77 15 60 52   ← 再对左右分别快速排序
       
第2趟(左28 35)：基准=28 → [28] 35
第2趟(右95 77 15 60 52)：基准=95 → 52 77 15 60 [95]

...最终：15 28 35 47 52 60 77 95
```

### 特性
| 项目 | 值 |
|------|-----|
| 平均时间复杂度 | **O(n log n)** |
| 最坏时间复杂度 | O(n²)（初始有序时） |
| 空间复杂度 | O(log n)（递归栈） |
| 稳定性 | ❌ **不稳定** |
| **特点** | 平均最快的排序算法 |

## 四、基数排序（Radix Sort）

**思想**：依次按个位、十位、百位、千位...分类，分一次收集一次，直到最高位。

```c
#define MAXSIZE 100

// 获取数组中的最大值
int getMax(int data[], int n)
{
    int max = data[0];
    for(int i = 1; i < n; i++)
        if(data[i] > max)
            max = data[i];
    return max;
}

// 基数排序
void radixSort(int data[], int n)
{
    int bucket[10][MAXSIZE];  // 10个桶
    int count[10];            // 记录每个桶中的元素个数

    int max = getMax(data, n);

    // exp表示当前处理位数：1→个位，10→十位，100→百位
    for(int exp = 1; max / exp > 0; exp *= 10)
    {
        // 桶计数初始化
        for(int i = 0; i < 10; i++)
            count[i] = 0;

        // 将元素放入对应桶中
        for(int i = 0; i < n; i++)
        {
            int digit = (data[i] / exp) % 10;
            bucket[digit][count[digit]] = data[i];
            count[digit]++;
        }

        // 按桶顺序收集元素
        int index = 0;
        for(int i = 0; i < 10; i++)
            for(int j = 0; j < count[i]; j++)
                data[index++] = bucket[i][j];
    }
}
```

### 执行过程示例
```
初始：53 3 542 748 14 214

按个位：542(2) 53(3) 14(4) 214(4) 748(8) 3(3)
收集：542 53 3 14 214 748

按十位：3(0) 14(1) 214(1) 542(4) 748(4) 53(5)
收集：3 14 214 542 748 53

按百位：3(0) 14(0) 53(0) 214(2) 542(5) 748(7)
收集：3 14 53 214 542 748 ✅
```

### 特性
| 项目 | 值 |
|------|-----|
| 时间复杂度 | O(d × (n + r)) |
| 空间复杂度 | O(r)（r为基数，如10） |
| 稳定性 | ✅ **稳定** |
| **特点** | 不需要比较，适合整数排序 |

## 五、四种排序对比

| 排序算法 | 平均时间 | 最坏时间 | 空间 | 稳定性 |
|---------|---------|---------|------|-------|
| **希尔排序** | O(n^1.3) | O(n²) | O(1) | ❌ 不稳定 |
| **归并排序** | **O(n log n)** | O(n log n) | **O(n)** | ✅ **稳定** |
| **快速排序** | **O(n log n)** | O(n²) | O(log n) | ❌ 不稳定 |
| **基数排序** | O(d(n+r)) | O(d(n+r)) | O(r) | ✅ 稳定 |

## 六、期末考试考点

### 📌 核心考点
1. **快速排序一趟划分结果**（必考大题）
2. **归并排序每趟归并后的序列**
3. **希尔排序每趟后的序列**
4. **快速排序最坏/最好情况**
5. **排序算法的选择**：
   - n较大，随机分布 → 快速排序
   - n较大，要求稳定 → **归并排序**
   - n较大，只取前k个最大/最小 → **堆排序**
   - 基本有序 → 直接插入排序

### 📌 常见题型
- **选择题**：时间复杂度、稳定性、适用场景
- **填空题**：写出每趟排序后的序列
- **简答题**：算法原理描述、优劣对比
- **算法题**：Partition函数实现

### 📌 易错点
| 易错点 | 说明 |
|--------|------|
| 快速排序最坏O(n²) | 初始有序时退化为冒泡 |
| 归并排序空间O(n) | 需要辅助数组 |
| 快速排序不稳定 | 基准元素交换破坏稳定性 |
| 希尔排序步长选择 | 最后必须为1 |
